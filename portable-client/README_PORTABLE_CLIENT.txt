Voice Punish Portable Client

Usage
1. Extract the whole zip to any folder.
2. Open "Plain Craft Launcher 2.exe".
3. In PCL, start the bundled instance "1.21.11-Fabric 0.19.1".
4. Wait for the ASR HUD to show "ASR: ready" on first run.

Notes
- This package includes the bundled Java runtime.
- This package includes the offline Chinese ASR runtime and model.
- The target platform is Windows x64.
- The player should still use their own Minecraft login in PCL if needed.
